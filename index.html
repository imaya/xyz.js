XYZ Loader in JavaScirpt

JavaScript で RPGツクール2000, RPGツクール2003 の独自画像形式である XYZ 画像の表示を可能にします。 xyz.min.js を読み込んだ後、CanvasTool.XyzLoad という function を呼び出す事で canvas 要素に描画します。

たとえば、XMLHttpRequest で受信した XYZ 画像データを canvas に描画するのは以下のようにします。

var xhr = new XMLHttpRequest();
var url = "sample.xyz";

xhr.responseType = "arraybuffer";
xhr.addEventListener('load', function(eve) {
  // new canvas element
  var canvas = document.createElement('canvas');

  // draw image
  CanvasTool.XyzLoad(new Uint8Array(xhr.response), canvas);

  // append to document body
  document.body.appendChild(canvas);
}, false);

xhr.open('GET', url, true);
xhr.send();
